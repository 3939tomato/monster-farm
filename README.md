# モンスター生態系シミュレーター
Geminiと一緒に作った、ドット絵モンスターの生態系シミュレーターです。

## 遊び方
- 画面をドラッグして移動
- モンスターをクリックで詳細確認
- エディターで自分だけのモンスターを放流！
- 運が良ければ突然変異個体が現れるかも！

## 注意点
- 注目追従がONになっていると画面が動かせません
- モンスターの数が多くなると重たくなります
- 同じ名前のモンスターを複数作らないでください
- 最初からスピードは12あります

## 属性について
- 火、水、木、ノーマル、闇、光　
- 火は木に強く、水に弱い　　ノーマルは光に強く、闇に弱い
- 水は火に強く、木に弱い　　闇はノーマルに強く、光に弱い
- 木は水に強く、火に弱い　　光は闇に強く、ノーマルに弱い
